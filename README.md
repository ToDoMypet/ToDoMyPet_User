# To Do My Pet - 유저 서비스

## 서비스 개요
1) 주제: 육성 시뮬레이션 게이미피케이션을 결합한 투두리스트
2) 목적: 사용자가 투두리스트로 목표를 작성하고 이를 하나하나 달성해가면서 자신의 성장을 캐릭터를 키움으로써 직관적으로 확인할 수 있다. 이를 통해 투두리스트를 달성하려는 목표 의식의 동기를 가질 수 있다.
3) 컨셉: 어제보다 오늘 더 성장한 ‘나’ + 게임으로 즐기는 투두리스트
